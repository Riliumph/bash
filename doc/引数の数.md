# 引数

シェルスクリプトでは、特殊変数を使って引数を取得する場合がある。

- `$@`
- `$*`

`$@`は、それぞれの要素を個別に参照することができる。  
対して、`$*`はすべての引数を連結した一つの文字列として受け取る。

## サンプル

```bash
#!/bin/bash

function echo_arg() {
  echo "num of arg: "$#
  echo "first arg : "$1
  echo "second arg: "$2
}

echo "case \$@"
echo_arg "$@"
echo "case \$*"
echo_arg "$*"
```

```terminal
$ ./test.bash arg1 arg2
case $@
num of arg: 2
first arg : arg1
second arg: arg2
case $*
num of arg: 1
first arg : arg1 arg2
second arg: 
```
